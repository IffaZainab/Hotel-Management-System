# The Platinum Court - Hotel Management System

A Python-based desktop Hotel Management System simulating 
real-world hotel operations through a feature-rich GUI.

## Features
- Animated splash screen and centralized main dashboard
- Room booking with images and pricing
- Event hall booking (weddings, birthdays, board meetings)
- Restaurant services: buffet bookings and table reservations
- Laundry service request forms
- Guest information collection with input validations
- Simulated payment via JazzCash, EasyPaisa, and card
- Automatic receipt generation
- About Us gallery with hotel showcase and contact details

## Tech Stack
Python | OOP | File Handling | GUI
